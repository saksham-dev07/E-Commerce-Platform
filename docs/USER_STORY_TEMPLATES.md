# User Stories Templates

## User Story Template
```
As a [type of user],
I want [some goal]
So that [some reason/benefit]

Acceptance Criteria:
- [ ] Given [context], when [action], then [expected outcome]
- [ ] Given [context], when [action], then [expected outcome]

Definition of Ready Checklist:
- [ ] Story is written in user language
- [ ] Acceptance criteria are clear and testable
- [ ] Story is sized (story points assigned)
- [ ] Dependencies identified
- [ ] UI/UX requirements clarified
- [ ] Technical approach agreed upon

Definition of Done Checklist:
- [ ] Code implemented and peer reviewed
- [ ] Unit tests written (>80% coverage)
- [ ] Integration tests pass
- [ ] UI is responsive
- [ ] Accessibility standards met
- [ ] Documentation updated
- [ ] Deployed to staging
- [ ] Product Owner accepted
- [ ] Performance criteria met
```

## Epic Template
```
**Epic Name**: [Brief, descriptive name]
**Epic ID**: [AUTO-GENERATED]

**Business Value**: Why is this epic important?

**User Personas**: Who will benefit from this epic?

**Success Criteria**: How will we measure success?
- Metric 1: Target value
- Metric 2: Target value

**Dependencies**: What other work must be completed?

**Risks**: What could prevent success?

**User Stories**: List of stories in this epic
- Story 1
- Story 2
- Story 3

**Timeline**: Expected completion timeframe
```
